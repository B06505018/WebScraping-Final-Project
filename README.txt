## 高齡生活大進擊  

以高齡人士的生活作為藍本的文字冒險遊戲。  

Team 8

### 遊玩方式
1. 滑鼠點擊【開始遊戲】，遊戲即開始。
2. 開始遊戲後，按【 Enter 】進行劇情。  
3. 選項出現時，請以滑鼠點擊任一選項。  
4. 死亡即是結局。已知結局有 3 個。  
5. 若要重新遊玩，請重整視窗。  

### 遊玩須知  
1. 完整劇本目前尚未呈現於成品，為顧及玩家的遊玩體驗，建議玩家選擇女性／無配偶／租屋之條件進行遊戲。  
2. 未完成的路線將會出現「未完待續」之提示。
